# References {#references}

ACAV uses and builds on several open-source technologies and project resources.

## Core Technologies

- [LLVM Project](https://llvm.org/) - compiler infrastructure.
- [Clang](https://clang.llvm.org/) - C/C++ compiler frontend and tooling
  libraries.
- [Qt Framework](https://www.qt.io/) - cross-platform GUI toolkit.

## Testing Infrastructure

- [LLVM Lit](https://llvm.org/docs/CommandGuide/lit.html) - LLVM Integrated
  Tester for integration tests.
- [FileCheck](https://llvm.org/docs/CommandGuide/FileCheck.html) - LLVM's
  pattern-matching file verifier.
- [Catch2](https://github.com/catchorg/Catch2) - C++ unit-testing framework.

## Related Resources

- [Clang Libraries Companion](https://github.com/mdadams/clang_libraries_companion)
  by Michael Adams - examples and utilities for working with Clang libraries.
- [Michael Adams' Packages](https://github.com/mdadams?tab=packages) -
  prebuilt LLVM/Clang packages that can support ACAV development and testing.
- [String interning](https://en.wikipedia.org/wiki/String_interning) - a
  technique used in ACAV for efficient string storage and comparison.
